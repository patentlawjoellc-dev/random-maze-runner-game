# Random Maze Runner

A browser-based 2D maze game built with React + Vite. Navigate from Start to End through a randomly generated, always-solvable maze.

## Gameplay

- Use arrow keys or WASD to move.
- Reach the End tile to complete the level.
- Each new level generates a new maze and gradually increases difficulty.
- The game tracks level, moves, best moves per level, and elapsed time.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

This is a static client-side app deployable to Vercel.
